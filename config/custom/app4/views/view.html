A little bit of extra code inside your view's route:

res.locals.breadcrumbs = {
    links: [
        {url: '/', text: res.locals.string.home},
        {active: true, text: 'Breadcrumbs'}
    ]
};

res.locals.breadcrumb is a variable that holds the data about admin's breadcrumbs.
This variable is part of the admin's UI so it's rendered outside of your custom template.
