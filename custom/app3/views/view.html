I'm styled through custom local stylesheet

This custom view has its own custom css and js files but this time instead of adding them inside the custom.json file we're adding them inside the template itself.

This makes them visible only for this view, you can inspect the document and see that they're not included inside the head tag of the document.

Remember that you still have to define the absolute path to your public directory inside the custom.json file.

Each of your custom files should have unique name among all included files even if they are in different directories!

Express and Nginx serve the static files on first matched basis.
